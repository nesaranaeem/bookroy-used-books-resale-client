# BookRoy - Resale Your Books

BookRoy is a platform to resell books.

## Features & Overview

- Built with React JS
- Responsive Design
- Multi Vendor System (User, Seller, Admin)
- Many Admin Functions
- Stripe Payment Gateway
- JWT Token
- Beautiful Cards
- Blog Page
- Error Page
- And Many More...
